# mac 常用命令
## terminal
	command + k //清屏
## safari
	command + d //添加书签
	shift + command + l //显示隐藏书签栏
	shift + command + b //显示隐藏收藏栏
## macOS
	option + command + d //显示隐藏dock
	control + command + f //全屏显示应用程序
	
## finder
	shift + command + a //应用程序
	shift + command + h //个人
	shift + command + r //隔空投送
	shift + command + c //电脑
